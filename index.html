Resumen de la página

La página usa div para agrupar secciones y span para resaltar texto. La imagen se creó en SVG y se insertó con la etiqueta img. La tabla muestra pasos/requisitos para obtener el trabajo. En el CSS se aplican width, height, background, color, margin y padding, con selectores por id (#resumen) y por clase (.section, .resaltado).
